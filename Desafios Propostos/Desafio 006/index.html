  • Links Internos e externos

    • Para abrir link internos abra uma outra pagina @.html(ou outra qualquer, menos a como nome original senão você vai se confudir ) e coloque na mesma pasta da do original

    Para links internos dentro (uma pasta dentro de uma pasta ) utilize o ../ mais o nome do arquivo na pasta

    Para links internos não existe a rel="external" mas existe a rel="next"(para abrir numa nova guia) e para o link que vai ser aberto utilize a rel="prev"

    Para um link onde voce não conhece o patrocinador utilize a target= "_blank" e a rel= "nofollow", significa que eu não dou o aval para o tipo de conteudo do site ou blog

    Dentro das Listas de Definições temos:

    • < dt > que é o termo
    • < dd > que é a descrição
    • < dl > que é como a lista fica no formato (externo)
