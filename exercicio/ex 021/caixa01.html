Exemplo de caixa box-level

Paragráfos tambem são exemplos de box level, mas os Links são exemplos de caixas inline level vamos ver como tudo funciona

Exemplo de shorthand com Padding:

  1. TOP (CIMA)
  2. RIGTH (DIREITA)
  3. BOTTOM (BAIXO)
  4. LEFT (ESQUERDA)

Nesse caso quando temos os quatro valores definidos, mas quando temos 2 o pimeiro é para cima e para baixo, e o segundo é para direita e esquerda, quando tem um valor somente ele vale para todos os lados
