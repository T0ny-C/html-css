Imagem dinânima

Tente abrir esse site em varios dispositivos diferentes ou simplesmente aumente e diminua o tamanho da janela do seu navegador.

A tag utilizada para adicionar imagens no seu site é < picture >
E em seguida utilize o < source: media type > para as imagens se adequar para o visitante dependendo de qual aparelho móvel ele está acessando

Seguindo a ordem das imagens de cima para baixo e da maior para menor

Reproduzindo Audios

Vamos aprender a reproduzir audios m html5.

Uma forma de adicionar audios é quase parecido da forma acima mas desse ultima tentativa é voce mesmo que vai digitar os sources e os types. Veja:

Atributos da tag source (mais utilizados)

  • TYPE: vai indicar o media type da imagem que usamos
  • SRCSET: vai configurar o nome da imagem que sera carregada quando o tamanho indicado for atingido
  • MEDIA: indica o tamanho maximo a ser considerado para carregar a imagem indicada no atributo SRCSET

O gustavo guanabara não recomenda utilizar arquivos WAVE porque eles são muito mas muito pesados e isso difilculta o acesso de pessoas no site
